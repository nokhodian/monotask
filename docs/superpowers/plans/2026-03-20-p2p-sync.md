# P2P Sync Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement automatic libp2p-based P2P board sync so boards in a Space sync in real-time across local networks and the internet with no server.

**Architecture:** libp2p 0.53 Swarm with mDNS (local) + Kademlia DHT (internet) discovery, Circuit Relay + DCUtR for NAT traversal, and a custom `/monotask/board-sync/1.0.0` request-response protocol carrying Automerge sync messages. Ed25519 identity bridges directly to libp2p PeerId via raw bytes.

**Tech Stack:** `libp2p 0.53`, `automerge 0.5` (sync module), `tokio`, `ciborium`, Rust workspace

**Spec:** `docs/superpowers/specs/2026-03-20-p2p-sync-design.md`

#[derive(Debug, Error)]
pub enum NetError {
    #[error("libp2p error: {0}")]
    Libp2p(String),
    #[error("storage error: {0}")]
    Storage(#[from] kanban_storage::StorageError),
    #[error("sync error: {0}")]
    Sync(String),
    #[error("handshake rejected: {0}")]
    Rejected(String),
    #[error("serialization error: {0}")]
    Serialization(String),
}

/// Configuration for the network node.
#[derive(Clone, Debug)]
pub struct NetConfig {
    /// UDP/TCP port to listen on. 0 = OS-assigned. Default: 7272.
    pub listen_port: u16,
    /// Data directory — port is saved here as `net.port` after bind.
    pub data_dir: PathBuf,
}

impl Default for NetConfig {
    fn default() -> Self {
        Self {
            listen_port: 7272,
            data_dir: PathBuf::from("."),
        }
    }
}

/// Events emitted by the network layer to callers.
#[derive(Debug, Clone)]
pub enum NetEvent {
    PeerConnected    { peer_id: String },
    PeerDisconnected { peer_id: String },
    BoardSynced      { board_id: String, peer_id: String },
    SyncError        { board_id: String, error: String },
}

/// Commands sent from callers into the network task.
#[derive(Debug)]
pub(crate) enum NetCommand {
    AnnounceSpaces { space_ids: Vec<String> },
    TriggerSync    { board_id: String },
    Stop,
}

/// Handle to the background network task.
pub struct NetworkHandle {
    pub(crate) cmd_tx: mpsc::Sender<NetCommand>,
    /// Receive sync events. Poll this in your event loop.
    pub event_rx: mpsc::Receiver<NetEvent>,
}

impl NetworkHandle {
    /// Start the network node in a background tokio task.
    ///
    /// `storage` must be `Arc<Mutex<Storage>>` because `rusqlite::Connection`
    /// is `!Send`. The lock is acquired only for load/save, never held across
    /// await points.
    pub async fn start(
        config: NetConfig,
        storage: Arc<Mutex<Storage>>,
        identity_bytes: [u8; 32],
    ) -> Result<Self, NetError> {
        let (cmd_tx, cmd_rx) = mpsc::channel::<NetCommand>(64);
        let (event_tx, event_rx) = mpsc::channel::<NetEvent>(256);

        tokio::spawn(swarm::run(config, storage, identity_bytes, cmd_rx, event_tx));

        Ok(Self { cmd_tx, event_rx })
    }

    /// Tell the node which Spaces this peer belongs to (re-call after join/leave).
    pub async fn announce_spaces(&self, space_ids: Vec<String>) {
        let _ = self.cmd_tx.send(NetCommand::AnnounceSpaces { space_ids }).await;
    }

    /// Trigger immediate sync for a board (debounced inside the swarm task).
    pub async fn trigger_sync(&self, board_id: String) {
        let _ = self.cmd_tx.send(NetCommand::TriggerSync { board_id }).await;
    }

    /// Gracefully stop the network task.
    pub async fn stop(&self) {
        let _ = self.cmd_tx.send(NetCommand::Stop).await;
    }
}

use std::io;
use libp2p::request_response;
use serde::{Deserialize, Serialize};

pub const PROTOCOL_NAME: &str = "/monotask/board-sync/1.0.0";

#[derive(Debug, Clone, Serialize, Deserialize)]
pub enum SyncRequest {
    /// Prove Space membership and share which boards this peer has.
    Hello {
        space_id: String,
        board_ids: Vec<String>,
        /// Ed25519 signature over `space_id.as_bytes()`.
        signature: Vec<u8>,
    },
    /// One round of Automerge sync for a board.
    BoardSync {
        board_id: String,
        /// `automerge::sync::Message::encode()` output.
        sync_message: Vec<u8>,
    },
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub enum SyncResponse {
    /// Hello accepted; here are the responder's board IDs in this Space.
    HelloAck { board_ids: Vec<String> },
    /// One round of Automerge sync in reply. `None` = this side has converged.
    BoardSync {
        board_id: String,
        sync_message: Option<Vec<u8>>,
    },
    /// Rejected: not in same Space, bad signature, or member is kicked.
    Rejected { reason: String },
}

/// CBOR codec for request_response::Behaviour.
#[derive(Debug, Clone, Default)]
pub struct MonotaskCodec;

#[async_trait::async_trait]
impl request_response::Codec for MonotaskCodec {
    type Protocol = String;
    type Request  = SyncRequest;
    type Response = SyncResponse;

    async fn read_request<T>(&mut self, _: &String, io: &mut T)
        -> io::Result<SyncRequest>
    where T: tokio::io::AsyncRead + Unpin + Send
    {
        use tokio::io::AsyncReadExt;
        let len = read_u32(io).await?;
        let mut buf = vec![0u8; len as usize];
        io.read_exact(&mut buf).await?;
        ciborium::from_reader(buf.as_slice())
            .map_err(|e| io::Error::new(io::ErrorKind::InvalidData, e.to_string()))
    }

    async fn read_response<T>(&mut self, _: &String, io: &mut T)
        -> io::Result<SyncResponse>
    where T: tokio::io::AsyncRead + Unpin + Send
    {
        use tokio::io::AsyncReadExt;
        let len = read_u32(io).await?;
        let mut buf = vec![0u8; len as usize];
        io.read_exact(&mut buf).await?;
        ciborium::from_reader(buf.as_slice())
            .map_err(|e| io::Error::new(io::ErrorKind::InvalidData, e.to_string()))
    }

    async fn write_request<T>(&mut self, _: &String, io: &mut T, req: SyncRequest)
        -> io::Result<()>
    where T: tokio::io::AsyncWrite + Unpin + Send
    {
        let mut buf = Vec::new();
        ciborium::into_writer(&req, &mut buf)
            .map_err(|e| io::Error::new(io::ErrorKind::InvalidData, e.to_string()))?;
        write_u32(io, buf.len() as u32).await?;
        use tokio::io::AsyncWriteExt;
        io.write_all(&buf).await
    }

    async fn write_response<T>(&mut self, _: &String, io: &mut T, res: SyncResponse)
        -> io::Result<()>
    where T: tokio::io::AsyncWrite + Unpin + Send
    {
        let mut buf = Vec::new();
        ciborium::into_writer(&res, &mut buf)
            .map_err(|e| io::Error::new(io::ErrorKind::InvalidData, e.to_string()))?;
        write_u32(io, buf.len() as u32).await?;
        use tokio::io::AsyncWriteExt;
        io.write_all(&buf).await
    }
}

async fn read_u32<T: tokio::io::AsyncRead + Unpin>(io: &mut T) -> io::Result<u32> {
    use tokio::io::AsyncReadExt;
    let mut buf = [0u8; 4];
    io.read_exact(&mut buf).await?;
    Ok(u32::from_be_bytes(buf))
}

async fn write_u32<T: tokio::io::AsyncWrite + Unpin>(io: &mut T, v: u32) -> io::Result<()> {
    use tokio::io::AsyncWriteExt;
    io.write_all(&v.to_be_bytes()).await
}

    }

    fn cbor_roundtrip_response(res: SyncResponse) -> SyncResponse {
        let mut buf = Vec::new();
        ciborium::into_writer(&res, &mut buf).unwrap();
        ciborium::from_reader(buf.as_slice()).unwrap()
    }

    #[test]
    fn serialize_hello_roundtrip() {

    autonat, dcutr, identify, kad, mdns, relay, request_response, swarm::NetworkBehaviour,
};
use crate::sync_protocol::{MonotaskCodec, SyncRequest, SyncResponse, PROTOCOL_NAME};

/// All libp2p behaviours composed into one.
/// Order matters: `identify` must be listed first — Kademlia and DCUtR depend on it.
#[derive(NetworkBehaviour)]
pub struct ComposedBehaviour {
    pub identify:          identify::Behaviour,
    pub mdns:              mdns::tokio::Behaviour,
    pub kademlia:          kad::Behaviour<kad::store::MemoryStore>,
    pub relay_client:      relay::client::Behaviour,
    pub dcutr:             dcutr::Behaviour,
    pub autonat:           autonat::Behaviour,
    pub sync:              request_response::Behaviour<MonotaskCodec>,
}

impl ComposedBehaviour {
    pub fn new(
        local_key: &libp2p::identity::Keypair,
        relay_behaviour: relay::client::Behaviour,
    ) -> Result<Self, Box<dyn std::error::Error>> {
        let local_peer_id = local_key.public().to_peer_id();

        let identify = identify::Behaviour::new(
            identify::Config::new("/monotask/1.0.0".into(), local_key.public())
                .with_push_listen_addr_updates(true),
        );

        let mdns = mdns::tokio::Behaviour::new(
            mdns::Config::default(),
            local_peer_id,
        )?;

        let mut kademlia = kad::Behaviour::new(
            local_peer_id,
            kad::store::MemoryStore::new(local_peer_id),
        );
        kademlia.set_mode(Some(kad::Mode::Server));

        let dcutr = dcutr::Behaviour::new(local_peer_id);
        let autonat = autonat::Behaviour::new(local_peer_id, autonat::Config::default());

        let sync = request_response::Behaviour::new(
            vec![(PROTOCOL_NAME.to_string(), request_response::ProtocolSupport::Full)],
            request_response::Config::default(),
        );

        Ok(Self {
            identify,
            mdns,
            kademlia,
            relay_client: relay_behaviour,
            dcutr,
            autonat,
            sync,
        })
    }
}

use libp2p::kad::{self, RecordKey};
use sha2::{Digest, Sha256};

/// Derives the Kademlia record key for a Space.
/// Two peers use the same key → they find each other.
/// Format: SHA-256("monotask/space/{space_id_hyphenated_uuid}")
pub fn space_dht_key(space_id: &str) -> RecordKey {
    let input = format!("monotask/space/{space_id}");
    let hash = Sha256::digest(input.as_bytes());
    RecordKey::new(&hash)
}

/// Bootstrap node multiaddresses (public IPFS nodes).
pub fn bootstrap_peers() -> Vec<(libp2p::PeerId, libp2p::Multiaddr)> {
    [
        ("/dnsaddr/bootstrap.libp2p.io/p2p/QmNnooDu7bfjPFoTZYxMNLWUQJyrVwtbZg5gBMjTezGAJN",
         "QmNnooDu7bfjPFoTZYxMNLWUQJyrVwtbZg5gBMjTezGAJN"),
        ("/dnsaddr/bootstrap.libp2p.io/p2p/QmbLHAnMoJPWSCR5Zhtx6BHJX9KiKNN6tpvbUcqanj75Nb",
         "QmbLHAnMoJPWSCR5Zhtx6BHJX9KiKNN6tpvbUcqanj75Nb"),
    ]
    .iter()
    .filter_map(|(addr_str, peer_str)| {
        let addr: libp2p::Multiaddr = addr_str.parse().ok()?;
        let peer_id: libp2p::PeerId = peer_str.parse().ok()?;
        Some((peer_id, addr))
    })
    .collect()
}

/// Announce all Spaces to the Kademlia DHT.
pub fn announce_spaces(kademlia: &mut kad::Behaviour<kad::store::MemoryStore>, space_ids: &[String]) {
    for space_id in space_ids {
        let key = space_dht_key(space_id);
        kademlia.start_providing(key).ok();
    }
}

/// Query the DHT for peers in a Space.
pub fn query_space_peers(kademlia: &mut kad::Behaviour<kad::store::MemoryStore>, space_id: &str) {
    let key = space_dht_key(space_id);
    kademlia.get_providers(key);
}

async fn handle_swarm_event(
    event: libp2p::swarm::SwarmEvent<ComposedBehaviourEvent>,
    swarm: &mut libp2p::Swarm<ComposedBehaviour>,
    storage: &Arc<Mutex<Storage>>,
    pubkey_cache: &mut std::collections::HashMap<libp2p::PeerId, libp2p::identity::PublicKey>,
    my_spaces: &[String],
    identity_bytes: [u8; 32],
    event_tx: &tokio::sync::mpsc::Sender<NetEvent>,
) {
    use libp2p::{identify, mdns, kad, request_response, swarm::SwarmEvent};
    use crate::sync_protocol::{SyncRequest, SyncResponse};

    match event {
        // Cache pubkey from Identify
        SwarmEvent::Behaviour(ComposedBehaviourEvent::Identify(
            identify::Event::Received { peer_id, info }
        )) => {
            pubkey_cache.insert(peer_id, info.public_key);
            // Feed observed addresses back into Kademlia
            for addr in info.listen_addrs {
                swarm.behaviour_mut().kademlia.add_address(&peer_id, addr);
            }
        }

        // mDNS: new local peer
        SwarmEvent::Behaviour(ComposedBehaviourEvent::Mdns(
            mdns::Event::Discovered(peers)
        )) => {
            for (peer_id, addr) in peers {
                swarm.behaviour_mut().kademlia.add_address(&peer_id, addr);
                tracing::debug!("net: mDNS discovered {peer_id}");
            }
        }

        // Peer connected — initiate sync handshake for each Space
        SwarmEvent::ConnectionEstablished { peer_id, .. } => {
            let _ = event_tx.send(NetEvent::PeerConnected { peer_id: peer_id.to_base58() }).await;
            initiate_hello(swarm, storage, my_spaces, identity_bytes, peer_id).await;
        }

        // Peer disconnected
        SwarmEvent::ConnectionClosed { peer_id, .. } => {
            let _ = event_tx.send(NetEvent::PeerDisconnected { peer_id: peer_id.to_base58() }).await;
        }

        // Incoming sync request
        SwarmEvent::Behaviour(ComposedBehaviourEvent::Sync(
            request_response::Event::Message {
                peer,
                message: request_response::Message::Request { request, channel, .. },
            }
        )) => {
            let response = handle_sync_request(
                request, peer, storage, pubkey_cache, my_spaces, event_tx,
            ).await;
            swarm.behaviour_mut().sync.send_response(channel, response).ok();
        }

        // Outgoing sync response received
        SwarmEvent::Behaviour(ComposedBehaviourEvent::Sync(
            request_response::Event::Message {
                peer,
                message: request_response::Message::Response { response, .. },
            }
        )) => {
            handle_sync_response(response, peer, storage, event_tx).await;
        }

        // Kademlia providers found — dial them
        SwarmEvent::Behaviour(ComposedBehaviourEvent::Kademlia(
            kad::Event::OutboundQueryProgressed {
                result: kad::QueryResult::GetProviders(Ok(kad::GetProvidersOk::FoundProviders {
                    providers, ..
                })),
                ..
            }
        )) => {
            for provider in providers {
                if provider != *swarm.local_peer_id() {
                    tracing::debug!("net: DHT found provider {provider}, dialing");
                    swarm.dial(provider).ok();
                }
            }
        }

        _ => {}
    }
}

async fn initiate_hello(
    swarm: &mut libp2p::Swarm<ComposedBehaviour>,
    storage: &Arc<Mutex<Storage>>,
    my_spaces: &[String],
    identity_bytes: [u8; 32],
    peer_id: libp2p::PeerId,
) {
    use crate::sync_protocol::SyncRequest;
    use kanban_crypto::Identity;

    for space_id in my_spaces {
        // Get boards in this space
        let board_ids = {
            let guard = storage.lock().unwrap();
            kanban_storage::space::get_space_boards(guard.conn(), space_id)
                .unwrap_or_default()
        };

        let identity = Identity::from_secret_bytes(&identity_bytes);
        let signature = identity.sign(space_id.as_bytes());

        swarm.behaviour_mut().sync.send_request(
            &peer_id,
            SyncRequest::Hello {
                space_id: space_id.clone(),
                board_ids,
                signature,
            },
        );
    }
}

async fn handle_sync_request(
    request: crate::sync_protocol::SyncRequest,
    peer: libp2p::PeerId,
    storage: &Arc<Mutex<Storage>>,
    pubkey_cache: &std::collections::HashMap<libp2p::PeerId, libp2p::identity::PublicKey>,
    my_spaces: &[String],
    event_tx: &tokio::sync::mpsc::Sender<NetEvent>,
) -> crate::sync_protocol::SyncResponse {
    use crate::sync_protocol::{SyncRequest, SyncResponse};
    use kanban_crypto::Identity;

    match request {
        SyncRequest::Hello { space_id, board_ids: their_board_ids, signature } => {
            // Verify signature using pubkey from Identify cache
            let pubkey = match pubkey_cache.get(&peer) {
                Some(pk) => pk,
                None => return SyncResponse::Rejected { reason: "no pubkey cached".into() },
            };
            // try_into_ed25519() gives ed25519::PublicKey; to_bytes() returns [u8; 32]
            let pubkey_32: [u8; 32] = match pubkey.clone().try_into_ed25519() {
                Ok(pk) => pk.to_bytes(),
                Err(_) => return SyncResponse::Rejected { reason: "not ed25519".into() },
            };
            if Identity::verify(&pubkey_32, space_id.as_bytes(), &signature).is_err() {
                return SyncResponse::Rejected { reason: "bad signature".into() };
            }

            // Check space membership (not kicked)
            let pubkey_hex = hex::encode(pubkey_32);
            let is_member = {
                let guard = storage.lock().unwrap();
                kanban_storage::space::is_active_member(guard.conn(), &space_id, &pubkey_hex)
                    .unwrap_or(false)
            };
            if !is_member {
                return SyncResponse::Rejected { reason: "not a member".into() };
            }

            // Return our board IDs in this space
            let my_board_ids = {
                let guard = storage.lock().unwrap();
                kanban_storage::space::get_space_boards(guard.conn(), &space_id)
                    .unwrap_or_default()
            };
            SyncResponse::HelloAck { board_ids: my_board_ids }
        }

        SyncRequest::BoardSync { board_id, sync_message } => {
            match process_incoming_sync(board_id.clone(), sync_message, storage).await {
                Ok(reply_msg) => {
                    let _ = event_tx.send(NetEvent::BoardSynced {
                        board_id: board_id.clone(),
                        peer_id: peer.to_base58(),
                    }).await;
                    SyncResponse::BoardSync { board_id, sync_message: reply_msg }
                }
                Err(e) => SyncResponse::Rejected { reason: e.to_string() },
            }
        }
    }
}

async fn process_incoming_sync(
    board_id: String,
    sync_message: Vec<u8>,
    storage: &Arc<Mutex<Storage>>,
) -> Result<Option<Vec<u8>>, crate::NetError> {
    use automerge::{AutoCommit, sync as am_sync};
    use kanban_core::init_doc;

    let msg = am_sync::Message::decode(&sync_message)
        .map_err(|e| crate::NetError::Sync(e.to_string()))?;

    let (mut doc, mut sync_state) = {
        let mut guard = storage.lock().unwrap();
        let doc = match guard.load_board(&board_id) {
            Ok(d) => d,
            Err(_) => {
                let mut d = AutoCommit::new();
                init_doc(&mut d).map_err(|e| crate::NetError::Sync(e.to_string()))?;
                d
            }
        };
        (doc, am_sync::SyncState::new())
    };

    doc.receive_sync_message(&mut sync_state, msg)
        .map_err(|e| crate::NetError::Sync(e.to_string()))?;

    {
        let mut guard = storage.lock().unwrap();
        guard.save_board(&board_id, &mut doc)
            .map_err(|e| crate::NetError::Storage(e))?;
    }

    let reply = doc.generate_sync_message(&mut sync_state)
        .map_err(|e| crate::NetError::Sync(e.to_string()))?;

    Ok(reply.map(|m| m.encode()))
}

async fn handle_sync_response(
    response: crate::sync_protocol::SyncResponse,
    peer: libp2p::PeerId,
    storage: &Arc<Mutex<Storage>>,
    event_tx: &tokio::sync::mpsc::Sender<NetEvent>,
) {
    use crate::sync_protocol::SyncResponse;
    match response {
        SyncResponse::HelloAck { board_ids: their_board_ids } => {
            tracing::debug!("net: HelloAck from {} boards={}", peer.to_base58(), their_board_ids.len());
            // Union: our boards + their boards (for boards we lack, create empty doc)
            let our_board_ids = {
                let guard = storage.lock().unwrap();
                guard.list_board_ids().unwrap_or_default()
            };
            let all_boards: std::collections::HashSet<String> =
                our_board_ids.into_iter().chain(their_board_ids).collect();

            for board_id in all_boards {
                let (mut doc, mut sync_state) = {
                    let mut guard = storage.lock().unwrap();
                    let doc = match guard.load_board(&board_id) {
                        Ok(d) => d,
                        Err(_) => {
                            let mut d = automerge::AutoCommit::new();
                            kanban_core::init_doc(&mut d).ok();
                            d
                        }
                    };
                    (doc, automerge::sync::SyncState::new())
                };
                if let Ok(Some(msg)) = doc.generate_sync_message(&mut sync_state) {
                    swarm.behaviour_mut().sync.send_request(
                        &peer,
                        crate::sync_protocol::SyncRequest::BoardSync {
                            board_id: board_id.clone(),
                            sync_message: msg.encode(),
                        },
                    );
                }
            }
        }
        SyncResponse::BoardSync { board_id, sync_message: Some(msg) } => {
            if let Err(e) = process_incoming_sync(board_id.clone(), msg, storage).await {
                let _ = event_tx.send(NetEvent::SyncError { board_id, error: e.to_string() }).await;
            } else {
                let _ = event_tx.send(NetEvent::BoardSynced { board_id, peer_id: peer.to_base58() }).await;
            }
        }
        SyncResponse::BoardSync { board_id: _, sync_message: None } => {
            tracing::debug!("net: peer {} converged", peer.to_base58());
        }
        SyncResponse::Rejected { reason } => {
            tracing::warn!("net: rejected by {}: {reason}", peer.to_base58());
        }
    }
}

async fn cmd_sync(
    data_dir: std::path::PathBuf,
    detach: bool,
    stop: bool,
    status: bool,
) -> anyhow::Result<()> {
    use kanban_net::{NetworkHandle, NetConfig, NetEvent};
    use kanban_storage::{Storage, space as space_store};
    use kanban_crypto::Identity;
    use std::sync::{Arc, Mutex};

    let pid_file = data_dir.join("sync.pid");

    if stop {
        let pid_str = std::fs::read_to_string(&pid_file)
            .map_err(|_| anyhow::anyhow!("sync daemon not running (no PID file)"))?;
        let pid: i32 = pid_str.trim().parse()?;
        unsafe { libc::kill(pid, libc::SIGTERM); }
        println!("Stopped sync daemon (PID {pid})");
        return Ok(());
    }

    if status {
        match std::fs::read_to_string(&pid_file) {
            Ok(pid) => println!("Sync daemon running (PID {})", pid.trim()),
            Err(_) => println!("Sync daemon not running"),
        }
        return Ok(());
    }

    if detach {
        // Re-exec self without --detach in background
        let exe = std::env::current_exe()?;
        let mut args: Vec<String> = std::env::args().collect();
        args.retain(|a| a != "--detach");
        let child = std::process::Command::new(exe)
            .args(&args[1..])
            .spawn()?;
        std::fs::write(&pid_file, child.id().to_string())?;
        println!("Sync daemon started (PID {})", child.id());
        return Ok(());
    }

    // Foreground: load identity and space IDs, then drop storage before re-opening as Arc<Mutex>
    let (identity_bytes, space_ids) = {
        let storage = Storage::open(&data_dir)?;
        let conn = storage.conn();
        space_store::get_profile(conn)?
            .ok_or_else(|| anyhow::anyhow!("No identity found. Run `monotask init` first."))?;
        let space_ids: Vec<String> = space_store::list_spaces(conn)?
            .into_iter().map(|s| s.id).collect();
        let key_path = data_dir.join("identity.key");
        let bytes = std::fs::read(&key_path)?;
        let identity_bytes: [u8; 32] = bytes
            .try_into()
            .map_err(|_| anyhow::anyhow!("invalid key file"))?;
        (identity_bytes, space_ids)
        // storage dropped here — conn borrow ends before Arc re-open
    };

    let storage = Arc::new(Mutex::new(Storage::open(&data_dir)?));
    let mut handle = NetworkHandle::start(
        NetConfig { listen_port: 7272, data_dir: data_dir.clone() },
        storage,
        identity_bytes,
    ).await?;

    handle.announce_spaces(space_ids).await;

    println!("Sync daemon running. Press Ctrl+C to stop.");
    loop {
        if let Some(event) = handle.event_rx.recv().await {
            match &event {
                NetEvent::PeerConnected { peer_id } =>
                    println!("✓ Peer connected: {peer_id}"),
                NetEvent::PeerDisconnected { peer_id } =>
                    println!("✗ Peer disconnected: {peer_id}"),
                NetEvent::BoardSynced { board_id, peer_id } =>
                    println!("⟳ Board {board_id} synced with {peer_id}"),
                NetEvent::SyncError { board_id, error } =>
                    println!("✗ Sync error for {board_id}: {error}"),
            }
        }
    }
}

## Notes for Implementers

- `swarm.rs` is the most complex file. If compile errors occur, read them carefully — libp2p 0.53's `SwarmBuilder` API and the `relay::client::Behaviour` integration are the most likely sources of friction. Check the libp2p 0.53 changelog if method signatures differ slightly.
- The integration test in Task 7 uses real mDNS on loopback — it may be flaky in CI environments that block multicast. Add `#[ignore]` if needed for CI, and run manually in development.
- `get_space_boards` and `is_active_member` (added to `kanban-storage/src/space.rs` in Task 6) must be added before Task 6 compiles — do that step before running `cargo check`.
- The Tauri startup in Task 9 uses `block_on` — ensure the Tauri app already runs inside a tokio runtime (it does, Tauri v2 uses tokio internally).
